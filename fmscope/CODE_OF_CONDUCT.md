# Code of Conduct

FMScope adopts the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. All contributors, maintainers, and users are
expected to abide by it in all project spaces — issues, pull requests,
discussions, and any direct communication tagged with the project.

## Reporting

Violations should be reported to the maintainer contact in
`CITATION.cff` (during the double-blind review period, the contact is
the anonymous mirror's issue tracker). Reports are handled
confidentially; reporters are protected from retaliation.

## Enforcement

The maintainers will follow the Contributor Covenant's standard
enforcement ladder: private warning → public warning → temporary ban →
permanent ban, depending on severity and recurrence.
